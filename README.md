🛒 Full Stack E-Commerce Store

A Full Stack E-Commerce Store built using Python for backend and HTML, CSS, JavaScript for frontend.
This project demonstrates a basic online shopping system where users can browse products, register/login,
and place orders.

# The project also uses local database storage to store product and user information.

---

📌 Project Overview

This project simulates an online e-commerce platform where users can:

- Browse available products
- View product details
- Register and login
- Add items to cart
- Place orders

The main objective of this project is to understand full stack web development including frontend, backend, and database integration.

---

🚀 Features

- 🛍 Product listing page
- 📄 Product details page
- 🛒 Shopping cart system
- 👤 User registration and login
- 📦 Order processing
- 💾 Local database storage for users and products

⚙ Installation & Setup

1️⃣ Clone the repository

git clone https://github.com/Abhishek-453/FullStack-Ecommerce-store.git

2️⃣ Navigate to the project folder

cd FullStack-Ecommerce-store

3️⃣ Install dependencies

pip install -r requirements.txt

4️⃣ Run the application

python app.py

5️⃣ Open in browser

http://localhost:8000
